## What is RailMind?

RailMind is a **dynamical system** where consciousness-like behavior emerges from **energy competition**, not brute-force compute.

Instead of static weights and fixed architectures, RailMind places autonomous agents ("Neural Flows") on a 3D energy grid. Each flow navigates under a **strict energy budget** â€” choosing where to go, what to reinforce, and what to abandon. The system's "intelligence" is the **trajectory itself**.

### Core Principles

| Principle | Description |
|-----------|-------------|
| **Energy Zero-Sum** | Fixed energy budget per cycle. No inflation. Every step costs. |
| **Dual-Timescale Memory** | Short-term "railroads" (fast, ephemeral) + long-term "rivers" (slow, geological). |
| **Fatigue-Driven Dispersion** | Visited nodes become repulsive â€” flows are forced to explore, never settle. |
| **Emergent Loops** | Self-sustaining circuits form spontaneously â€” the system's "working memory". |
